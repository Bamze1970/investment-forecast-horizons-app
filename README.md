# Investment Forecast Web App – Horizon PIE Edition

Това е обновеното уеб/PWA приложение за GitHub Pages.

## Какво има вътре
- Dashboard с overview карти
- Bottom navigation: Dashboard / Графики / Solana
- Големи бутони в Dashboard-а към отделните хоризонти
- Отделен екран за всеки хоризонт: Месец, Q1, Q2, Q3, Q4, 1Y, 2Y, 3Y, 4Y
- За всеки хоризонт: 3 PIE графики (Песимистичен / Основен / Оптимистичен)

## Как да качиш през GitHub
1. Разархивирай файла.
2. Качи всички файлове в root-а на GitHub repository.
3. Включи GitHub Pages от main branch и /(root).
4. Отвори линка на телефона.
5. В Chrome използвай Add to Home Screen / Install.
